## 核心结论

1. **H1a得到支持**：数据要素利用显著提升资本定价效率（降低股价延迟），DU_kw系数在1%水平显著
2. **信息供给效应占主导**：数据利用吸引分析师关注，改善信息环境
3. **资产配置渠道存在**：数据利用高的企业金融资产配置也更多
4. **入表新规短期增加信息处理负担**：DID正向系数（t=1.75*），但入表后投资组合alpha显著扩大（+2.08%**）
5. **所有异质性组均显著**：效应稳健，小企业和高分析师覆盖组效应更强
6. **数据开发能力是最关键子维度**：SHAP贡献占比39.7%
