## Remaining Issues (Round 2未修复)

### Issue A: 学者流水账的潜在残留 [MINOR BUT CRITICAL FOR VENUE]

**位置**：小节（二），第27行
**当前表述**：
```
这一规律的因果性已在多个市场中得到确认：
美国市场的断点识别设计揭示了分析师覆盖与股价延迟之间的因果关系（Hong等，2000），
中国市场的经验证据同样表明分析师覆盖对价格发现具有显著的促进作用（Yu，2008），
而增量信息披露的效果大小还取决于分析师的覆盖密度——分析师覆盖越高，
信息转换为价格的效率越高（黄俊和陈信元，2023）。
上述研究在识别策略和样本覆盖上各有侧重，但共同指向...
```

**问题分析**：虽然加入了统一的"共同指向"收尾，但中间三个逗号/破折号前的论述仍保留了Hong、Yu、黄俊陈信元三个分离的论证单元。与朱康(2025)《会计研究》参照范文对照，这类结构仍可被严格评审视为"多个学者主语的并列"而非完全的一体化论述。

**评审意见**：为了达到《会计研究》标准，建议进一步改为：
```
分析师覆盖对价格发现的加速效应已在多个市场中得到实证确认。
Hong等（2000）在美国市场通过断点识别设计揭示了分析师覆盖与股价延迟之间的因果关系，
其在中国市场的稳健性随后经由Yu（2008）和黄俊陈信元（2023）的研究得到验证。
这些研究虽在识别策略和样本覆盖上各有侧重，但共同指向同一结论：
分析师覆盖密度越高，信息向价格的转换效率越高。
```

这样改变的优势：(1)强化了"规律"而非"学者"的主语地位；(2)将三个研究置于同一规律的验证序列中，而非平行列举
