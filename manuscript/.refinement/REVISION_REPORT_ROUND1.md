### （一）数据要素利用与资本市场定价效率

#### 问题诊断

**框架先行不足（检查项1）**
- 原文第一段直接引入Ohlson（1995）理论框架，但缺少高层分析框架说明
- 未在开篇点明本节的核心逻辑链：会计计量偏差→信息不对称→定价效率

**学者主语过多（反模式检查）**
- 原文第五行："黄世忠等（2023）指出，数据资产在确认与计量层面..." 
- 以学者为主语，未体现机制/现象为主的表达原则

**文献对照缺失（检查项2）**
- 信息供给效应段落：Diamond和Verrecchia（1991）+ 黄俊和陈信元（2023）各自引用，但未形成对照
- 信息模糊效应段落：仅引Zhang（2006），缺少多篇文献对比

**证据质量评价不足（检查项4）**
- 论述"若信息供给效应为主导"的理由不够有力
- 未明确指出本文使用关键词法的优势相对于其他测量方式的局限性对比

#### 修订方案

**框架重构**
- 第一段改为分析框架导向：开篇说明"框架的可靠性建立于账面价值的代表程度"，后述数据资产表外存在破坏这一前提，形成"计量缺口→信息不对称→定价效率困境"的逻辑链

修订前：
```
依照Ohlson（1995）的剩余收益估值框架，企业内在价值由账面净资产与未来超额收益的贴现值共同决定，
账面价值的准确性直接影响企业价值评估的可靠性。数据资产的大规模表外存在使这一框架面临新的挑战...
```

修订后：
```
在Ohlson（1995）的剩余收益估值框架中，企业内在价值由账面净资产与未来超额收益的贴现值共同决定。
这个框架的可靠性建立于一个关键前提：账面价值对企业经济价值的代表程度。但数据资产的大规模表外存在
打破了这一前提。数据资产的会计计量偏差导致账面价值系统性低估，进而制造了投资者对企业真实价值的信息
不对称。这种信息不对称可能通过两条相反方向的路径影响资本定价效率，其净效果需通过实证检验确定。
```

**消除学者主语**

- 修订："黄世忠等（2023）在对数据资产确认与计量困难的分析中指出，历史成本模式与数据资源动态价值特征之间的矛盾..."

**文献对照加强**
- 信息供给效应段：Diamond和Verrecchia（1991）与黄俊和陈信元（2023）形成跨市场对照
  * 原表述"这一机制在中国市场同样成立"改为"Diamond和Verrecchia（1991）在一般框架中阐述了这一机制，黄俊和陈信元（2023）的中国市场证据进一步证实了增量信息披露能够加速私有信息向股票价格的融合，表明这一规律具有跨市场的稳健性"

- 信息模糊效应段：单篇Zhang（2006）的引用方式改为"Zhang（2006）的研究表明"，避免过于突兀

**理由论证加强**
- 原文仅列三点理由
- 修订后在第三点处增加对"关键词频率法区分实质性利用与纯粹提及"这一本文特色的强调，突出相对于其他披露指标的优势

## 四、写作反模式的检查与消除

### 反模式：流水账式学者主语

**原文示例**：
```
黄世忠等（2023）指出，数据资产在确认与计量层面面临独特困难。
Diamond和Verrecchia（1991）认为可信的公开信息披露能够降低逆向选择程度。
Hong等（2000）的研究表明分析师覆盖与股价延迟之间的因果关系。
```

**修订原则**：
- 每段最多1句以学者为主语
- 其余引用嵌入为括号注脚
- 主语切换为机制、现象或概念

**修订示例**：

修订前：
```
黄世忠等（2023）指出，数据资产在确认与计量层面面临独特困难，历史成本模式无法反映数据资源的动态
价值特征，由此产生的会计计量偏差构成账面价值与市场价值背离的重要来源。
```

修订后：
```
黄世忠等（2023）在对数据资产确认与计量困难的分析中指出，历史成本模式与数据资源动态价值特征之间
的矛盾是导致会计计量偏差的根本原因，而这种偏差正是账面价值与市场价值背离的重要来源。
```

进一步优化：
```
历史成本模式与数据资源动态价值特征之间的矛盾是导致会计计量偏差的根本原因，而这种偏差正是账面
价值与市场价值背离的重要来源（黄世忠等，2023）。
```
