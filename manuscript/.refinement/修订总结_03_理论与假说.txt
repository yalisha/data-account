三、写作反模式的消除
================================================================================

反模式识别：流水账式学者主语

问题示例（修订前）：
  "黄世忠等（2023）指出，数据资产在确认与计量层面面临独特困难。
   Diamond和Verrecchia（1991）认为可信的公开信息披露能够降低逆向选择程度。
   Hong等（2000）的研究表明分析师覆盖与股价延迟之间的因果关系。"

修订原则：
  ✓ 每段最多1句以学者为主语
  ✓ 其余引用嵌入为括号注脚
  ✓ 主语切换为机制、现象或概念

修订示例：
  修订前：黄世忠等（2023）指出，数据资产在确认与计量层面面临独特困难...
  修订后：历史成本模式与数据资源动态价值特征之间的矛盾是导致会计计量偏差
         的根本原因，而这种偏差正是账面价值与市场价值背离的重要来源
         （黄世忠等，2023）。

修订数据：学者主语句数从8句削减至4句（-50%）

小节（一）数据要素利用与资本市场定价效率
────────────────────────────────────

主要问题：
  1. 框架不显：直接引理论，缺高层逻辑
  2. 学者主语："黄世忠等（2023）指出..."在开篇出现
  3. 文献对照缺失：两个效应各自引用，缺对比
  4. 理由论证：三点理由中第三点的关键词法优势未明确

修订亮点：
  ✓ 首段重构为"Ohlson框架前提→数据资产表外→计量缺口→信息不对称"
  ✓ 将"黄世忠等指出"改为"历史成本模式...矛盾是...原因"（机制为主语）
  ✓ 供给效应与模糊效应的对比通过"Diamond/黄俊跨市场对照"体现
  ✓ 在第三点理由中强调"关键词频率法区分实质性vs概念"的方法论优势

小节（二）传导机制
────────────────────────────────────

主要问题：
  1. 框架简略："两个维度"的定义和分类逻辑不够明确
  2. 文献层次关系不清：Hong/Yu/黄俊三篇文献各自的贡献差异未说明
  3. 朱康vs李姝："不同视角"的具体差异未阐释
  4. 机构投资者段落："作用方向复杂"的复杂性根源未溯源

修订亮点：
  ✓ 新增"信息两阶段流动"框架说明（捕捉加工 vs 投资决策）
  ✓ 显式标注Hong的美国因果识别→Yu的中国验证→黄俊的条件效应
  ✓ 朱康/李姝的视角差异明确为"资本成本降低vs资产选择精准度"
  ✓ 机构投资者复杂性溯源至"投资者类型异质性"的理论根源
  ✓ 新增H2a明确标注为"第二个假说"，强化假说的正式性

小节（三）异质性视角
────────────────────────────────────

主要问题：
  1. 框架过渡句简略：未与H1-H2b形成递进关系
  2. 产权性质维度："大量文献表明"的空泛表述无具体引用
  3. 分析师维度：未明确说明这是对H2a的异质化分析
  4. 维度结尾不统一：三个维度的收尾方式不一致

修订亮点：
  ✓ 开篇增加"企业特征禀赋和市场环境影响总体效应"的过渡框架
  ✓ 产权维度改为直述"信息不对称与传导效率在产权结构下的特征差异"
  ✓ 分析师维度明确标注"核心传导机制之一在于...（H2a）"
  ✓ H3假说标注为"第四个假说"，与H1H2保持逻辑一致
  ✓ 三个维度的结尾统一：产权与行业维度均以"有待实证检验"结尾
