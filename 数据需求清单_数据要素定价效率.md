# 数据需求清单：数据要素利用、资产配置与资本定价效率

---

## 一、核心变量数据

### 1. 自变量：数据要素利用程度（LLM构建）

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 上市公司年报全文（PDF/TXT） | 巨潮资讯网 / CNINFO | 需要原始全文用于LLM语义分析 |
| 年报MD&A章节文本 | CSMAR文本数据库 / Wind | 管理层讨论与分析部分，信息密度最高 |
| 数据要素相关关键词词典 | 朱康（2025）附录 / 自建 | 作为LLM评分的参照基准 |

### 2. 因变量：资本定价效率

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 个股日收益率 | CSMAR / Wind | 计算股价同步性、股价延迟、方差比率 |
| 市场日收益率（沪深300/中证全指） | CSMAR / Wind | 市场模型基准 |
| 行业日收益率 | CSMAR | 按证监会行业分类 |
| 个股季度/年度财务数据 | CSMAR / Wind | 时序模型预测基本面价值的输入 |
| 分析师一致预期盈余 | CSMAR / Wind / 朝阳永续 | 用于构建基于预期的mispricing指标 |

### 3. 中间变量：资产配置结构

| 数据项 | 来源 | 说明 |
|--------|------|------|
| 交易性金融资产 | CSMAR资产负债表 | 金融资产配置的核心科目 |
| 可供出售金融资产 / 其他非流动金融资产 | CSMAR | 新旧准则口径均需 |
| 投资性房地产 | CSMAR | 金融化的另一种体现 |
| 固定资产、在建工程 | CSMAR | 实体投资的度量 |
| 无形资产（总额及明细） | CSMAR / 年报附注手工整理 | 需区分数据资产、软件、专利等 |
| 数据资源入表金额 | 手工整理（2024年起） | 2024年新规后披露的数据资产科目 |
| R&D投入 | CSMAR | 创新投资维度 |

## 七、数据获取优先级

第一批（立即可获取）：CSMAR全部财务数据、股票交易数据、公司治理数据、分析师预期数据

第三批（辅助数据）：宏观经济数据、城市互联网基础设施指数、大数据交易所设立时间
